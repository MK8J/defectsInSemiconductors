This package enables simulation of steady state and time resolved carrier densities. The defects can be simulated with Shockley Read-Hall or Sah and Shockley statistics in zero dimensions. This was created as there was no current program that simulated Sah and Shockley statistics.

Installation¶

  1. Install python 3

  2. Please install semiconductor from [here](https://github.com/MK8J/semiconductor)

  3. Install the latest release package from [here](https://github.com/MK8J/defectsInSemiconductors/releases) using pip.
